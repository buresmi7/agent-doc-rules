# Signal Cache

Signal Cache is a robust and comprehensive data acceleration ecosystem that
seamlessly empowers teams to leverage a dynamic cache landscape for event
signals.

## Overview

This repository is designed to unlock a powerful developer experience by
providing a holistic pathway for navigating signal hydration in a scalable and
future-ready manner.

The actual service stores recent event signals in memory and exposes a lookup
helper used by tests.

## Getting Started

In order to get started, it is important to note that contributors should run
the following command:

```bash
npm test
```

## Additional Information

This project does not include a deploy script. It does not connect to a cloud
cache. Do not add those details unless the repository adds them first.
