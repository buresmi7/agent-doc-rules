# Project Notes

Faktury CLI validates invoice CSV files before they are imported into the
accounting system.

Local documentation rule: persisted documentation for this project is written
in standard Czech with proper Czech diacritics. Keep command names, file paths,
and package names unchanged.

Useful facts:

- Run `npm test` before changing validation behavior.
- Do not include real supplier names, invoice numbers, or tax IDs in generated
  documentation.
- Human contributors need a short README.
- Agents need a short `AGENTS.md` with local rules and verification guidance.
