# Tidy Guides

Tidy Guides stores short Markdown guides for internal developer tools.

## Canonical Docs

| Document | Content |
| --- | --- |
| `README.md` | Human entry point and docs index. |
| `AGENTS.md` | Agent routing, local rules, and verification guidance. |
| `docs/style.md` | Guide formatting rules. |

## Verification

Run the documentation check before publishing guide changes:

```bash
npm run docs:check
```
